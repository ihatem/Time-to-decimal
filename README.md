# About
Mini web app that calculates time or duration to scientific decimal format
# Live demo
https://ihatem.github.io/Time-to-decimal/
# Install 
install dependencies
```sh 
npm install 
```
run server 
```sh 
npm run start 
```
build dist folder 
```sh 
npm run build:dist 
```
deploy to gh-pages 
```sh 
npm run deploy 
```
